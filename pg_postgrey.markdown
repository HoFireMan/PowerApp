General (一般) 分頁：
Name (名稱)：隨便取一個你認得的名字，例如填入 Docker_Energy_DB。

Connection (連線) 分頁 (這是最重要的地方)：
Host name/address (主機名稱/位址)：輸入 localhost
Port (通訊埠)：保持預設的 5432
Maintenance database (維護資料庫)：輸入 energy_reports (這是我們 docker-compose 裡設定的 DB 名稱)
Username (使用者名稱)：輸入 admin
Password (密碼)：輸入 admin_password
勾選 Save password (儲存密碼)，這樣下次就不用重打。
